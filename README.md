# Branching Out

MSIT Git Branching assignment. Demonstrates branches, merges (fast-forward only), pull requests, and a small interactive user-search program.

## Files

- `filter_users.py` - interactive search over a small in-memory user database. Asks the user which category to search by (`name`, `age`, `email`) and a query value, then prints matches. Includes substring matching for name and email, and operator queries (`>=30`, `<25`, ...) for age.
- `greeting.py` - `greet(name)` helper.
- `farewell.py` - `farewell(name)` helper.

## Run

```
python3 filter_users.py
```
